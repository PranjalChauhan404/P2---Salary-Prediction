💼 Salary Prediction using Linear Regression

📌 Overview

This project predicts employee salaries using the **Linear Regression** algorithm.

The model is trained on features such as:

- Age
- Gender
- Education
- Experience
- Location
- Job Title

The project demonstrates the complete Machine Learning workflow including data preprocessing, visualization, model training, evaluation, and prediction.

📂 Dataset

The dataset contains employee information including:

- Age
- Gender
- Education
- Experience
- Location
- Job Title
- Salary (Target)

🛠️ Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

📊 Data Preprocessing

- Removed null values
- Converted Age to integer
- Encoded Gender (Male/Female → 0/1)
- Applied One-Hot Encoding on categorical features
- Split dataset into training and testing sets


🤖 Model

- Linear Regression

📈 Model Performance

| Metric | Score |
|---------|-------|
| R² Score | **0.87** |
| MAE | **8157.90** |
| RMSE | **10295.45** |
| MSE | **105996277.66** |
